# Checklist 01: Legal Entity Setup & Regulatory Registration

## 1.1 Legal Entity
- [ ] Special purpose vehicle (SPV) or issuing entity incorporated
- [ ] Jurisdiction of incorporation selected and documented (with rationale)
- [ ] Beneficial ownership structure documented
- [ ] Registered agent appointed
- [ ] Operating agreement / articles of incorporation executed

## 1.2 Regulatory Status
- [ ] Token classification determined: **security token / ART / utility / other**
- [ ] Securities law analysis obtained from qualified counsel (per offering jurisdiction)
- [ ] Regulatory exemption identified:
  - [ ] Reg D 506(b) / 506(c) (US accredited investors)
  - [ ] Reg S (non-US persons)
  - [ ] Reg A+ (up to $75M, US)
  - [ ] MiCA authorization (EU)
  - [ ] Other: `_______________`
- [ ] Regulatory counsel engaged

## 1.3 Broker-Dealer / Placement Agent
- [ ] Determine if broker-dealer registration required
- [ ] If yes: registered BD engaged or registered status obtained
- [ ] Placement agent agreement executed (if applicable)

## 1.4 Transfer Agent
- [ ] Determine if registered transfer agent required (US: Section 17A of Exchange Act)
- [ ] Transfer agent engaged or exemption documented
- [ ] Smart contract compliance with transfer agent functions reviewed

## 1.5 AML/BSA Program (US)
- [ ] AML program established (if subject to BSA)
- [ ] Designated BSA officer appointed
- [ ] Customer Identification Program (CIP) documented
- [ ] Suspicious activity reporting procedures in place
