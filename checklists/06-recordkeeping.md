# Checklist 06: Books & Records

## 6.1 Transaction Records
- [ ] On-chain transaction log exported and archived regularly
- [ ] Off-chain transaction records maintained in WORM-compliant storage
- [ ] Retention period: minimum 6 years [FINRA 4511 / SEC 17a-4]
- [ ] Downloadable on demand confirmed

## 6.2 Investor Records
- [ ] Register of members / token holders maintained off-chain
- [ ] Cap table reconciled with on-chain balances at least monthly
- [ ] Investor onboarding documents archived
- [ ] Correspondence related to token offering archived

## 6.3 Compliance Records
- [ ] KYC/AML determinations archived per investor
- [ ] Transfer restriction determinations archived
- [ ] Audit trails for compliance module configuration changes retained
- [ ] Smart contract deployment records archived (deployment tx hash, audited bytecode)

## 6.4 Corporate Records
- [ ] Board / member resolutions related to token issuance archived
- [ ] Legal opinions archived
- [ ] Regulatory filings archived (Reg D Form D, MiCA white paper filing, etc.)

## 6.5 Storage & Access
- [ ] Primary storage: `[PROVIDER]` — WORM-compliant confirmed
- [ ] Backup storage: `[PROVIDER]`
- [ ] Access control: restricted to authorized personnel
- [ ] Auditor access: third-party auditor designated [SEC 17a-4]
- [ ] Index maintained for all archived records
