# RWA Compliance Checklist

> An open-source compliance checklist for real-world asset (RWA) token issuers. Covers KYC/AML, investor onboarding, transfer restrictions, and multi-jurisdiction regulatory mapping.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

---

## Who this is for

- **Token issuers** launching RWA offerings (real estate, private credit, commodities, funds)
- **Legal and compliance teams** structuring tokenized securities
- **Infrastructure providers** building ERC-3643 / T-REX platforms
- **Founders** navigating the compliance requirements of their first token offering

## How to use

1. Clone this repo or bookmark it
2. Work through checklists sequentially for a new offering
3. Use as a due diligence reference when reviewing third-party token offerings
4. Contribute gaps, corrections, or new jurisdiction coverage via PR
